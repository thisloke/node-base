## Quick usage

### 1) Get your copy
```
npx github:thisloke/node-base my-node-project
```

### 2) Start development
```
 npm run build:watch
 npm run test:watch
```

### 3) Build
```
 npm run build
```

### 4) Run
```
 npm run start
```
